# 总结webpack

## Webpack的基本概念

1. 入口文件entry

- 以某个文件为入口开始打包
- 分类
  - 单入口 String
    - 只会输出一个文件(SPA)
  - 多入口 Array / Object
    - Array 只会输出一个文件
    - Object 会输出多个文件
      - 多页面应用

2. 输出文件output

- 输出的文件

3. loader

4. 插件 plugin

5. mode

- 模式: 开发环境(development) 和生产环境(production)
